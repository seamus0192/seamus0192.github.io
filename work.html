Object Oriented 2D Maze Game

View Code on GitHub

This project is a game coded in Java using many object-oriented essentials and was developed by myself and a fellow student, Peter Pham. The premise of the game is that the user plays as a lasagna, running away from garfields. Points are gained by routing to the microwave, and the game is over once your health runs out, which happens when garfields catch up to you. However, health can also be gained by walking over lasagna ingredients (cheese, sauce). You can also eliminate garfields with bombs, allowing the user to survive once the game starts to get super hectic. There are different types of garield enemies, each which use a different pathfinding strategy in order to reach the player (respectivly being A Star Pathing, Random pathing, and Single Step pathing).
